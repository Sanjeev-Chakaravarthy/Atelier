# Atelier — Premium Productivity

Atelier is a premium digital workspace designed for high-output professionals. It blends structured, high-density utility with editorial sophistication to offer a seamless, minimal distraction task manager and productivity suite.

## 🚀 Features

- **High-Density Task Board**: Structured task management designed to minimize visual clutter and streamline execution workflows.
- **Productivity Rhythms & Analytics**: Visual tooltips and bar charts tracking deep work sessions.
- **Deep Work Focus Space**: Built-in Pomodoro/Focus timer with real-time browser desktop push notifications and synthesized Web Audio alarm chimes.
- **Personalized Workspace**: Secure Google OAuth authentication, default avatar synchronization, and custom profile timezone support.

## 🛠️ Stack

- **Frontend**: React (Vite), Tailwind CSS, Framer Motion, Lucide icons, React Big Calendar.
- **Backend**: Node.js, Express, MongoDB (Mongoose), JSON Web Tokens (JWT), Web Audio API, Google Identity Services.

## 📦 Getting Started

### 1. Prerequisites
- Node.js (v18+)
- MongoDB (local or Atlas cluster)

### 2. Installation
Clone the repository:
```bash
git clone https://github.com/Sanjeev-Chakaravarthy/Atelier.git
cd Atelier
```

Install root, client, and server dependencies:
```bash
npm install
cd client && npm install
cd ../server && npm install
```

### 3. Run Development Servers
From the root directory:
```bash
npm run dev
```
